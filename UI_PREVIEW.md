## Вкладки

### 1. Чат (основная)

- **История сообщений** в виде карточек (ассистент слева, пользователь справа)
- **Поле ввода** с кнопкой «Отправить»
- **Время** каждого сообщения

### 2. Логи агентов

- **Real‑time поток** логов OpenClaw Gateway (stdout/stderr)
- **Цветовое кодирование**: зелёный (stdout), красный (stderr)
- **Кнопки**: «Очистить», «Сохранить»

### 3. Файлы рабочего пространства

- **Список файлов** и папок из `~/.openclaw/workspace`
- **Иконки типов** (📄 для .md, 📁 для папок, 📊 для .pdf, etc.)
- **Кнопки**: «Обновить», «Открыть папку»
- **Двойной клик** для редактирования (будет открывать редактор)

### 4. Активные агенты

- **Карточки агентов** с статусом (зелёный/жёлтый/красный)
- **Информация**: ID, роль, задача
- **Форма создания** нового агента (имя + задача)

### 5. Настройки

- **OpenClaw**: автозапуск Gateway, путь к CLI, порт
- **Уведомления**: звук, всплывающие окна
- **Внешний вид**: тема (светлая/тёмная/авто), размер шрифта
- **Кнопки**: «Сохранить», «Сбросить»

## Стиль

- **Цветовая схема**: синий акцент (#3B82F6), серые фоны, белые панели
- **Шрифт**: Inter (современный, читаемый)
- **Иконки**: Font Awesome 6
- **Анимации**: плавные переходы между вкладками, hover‑эффекты
- **Адаптивность**: сетка Tailwind, работает от 1024px и выше
